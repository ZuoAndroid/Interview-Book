### 1. 下面两段代码的输出是什么？
第一段代码：

```python
a = 1

def fun(a):
    a = 2
	
fun(a)
print(a)
```
输出结果:
```
1
```

第二段代码：
```python
a = []

def fun(a):
    a.append(1)
fun(a)
print(a)
```

输出结果：

```
[1]
```
### 2. 写一个装饰器
```python
def singleton(cls, *args, **kw):
    instances = {}
    def getinstance():
        if cls not in instances:
            instances[cls] = cls(*args, **kw)
        return instances[cls]
    return getinstance

@singleton
class MyClass:
  ...
```
